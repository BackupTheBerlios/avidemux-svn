Dual audio track

1.Warning

    Avidemux can only handle one audio track internally.

    Nonetheless, there is a way to add a second audio track, with limited support and no processing at all on that second audio track.

2.Adding a second audio track

Restrictions 

    The second audio track must be CBR, that is MP3 CBR or AC3. Period.

Menu 

    To add a second audio track, use the Audio->Second Track->xxxx menu. After selecting it, use File -> Dual audio avi instead of File ->Save video.

Cutting a dual audio avi 

    As Avidemux cannot handle more than one audio stream internally we will have to play a bit .

    Open the avi, when prompted select the second audio track

    Set the A and B markers and use File -> save audio as /tmp/track11.mp3 for example.

    Reopen the same avi, but this time select the first audio track.

    Cut it, and save it as /tmp/avi1.avi.

    Reload it, and add /tmp/track11.mp3 as second track. Save using File -> Dual audio avi menu.

    Repeat for each part.
