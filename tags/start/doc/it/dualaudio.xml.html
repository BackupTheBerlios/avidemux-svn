Traccia audio Dual

1.Avviso

    Avidemux è in grado di operare internamente solo su una traccia audio.

    Tuttavia vi è un modo per aggiungere una seconda traccia audio, anche se con supporto limitato e senza poter essere processata.

2.Aggiungere una seconda traccia audio

Restrizioni 

    La seconda traccia audio deve essere in CBR, sia essa MP3 o AC3. Punto.

Menu 

    Per aggiungere una seconda traccia audio utilizzare il menu Misc->Second Track->xxxx. Dopo aver selezionato il file, utilizzare l'opzione File -> Dual audio avi invece di File -> save avi.

Tagliare un avi con dual audio 

    Dato che Avidemux non è in grado di operare internamente su più di un flusso audio, dovremo aggirare questo problema.

    Aprire il file avi, quando richiesto selezionare la seconda traccia audio.

    Posizionare i marcatori A e B e selezionare il menu File -> save audio digitando per esempio /tmp/track11.mp3.

    Riaprire lo stesso file avi, ma stavolta selezionare la prima traccia audio.

    Tagliare e salvare come /tmp/avi1.avi.

    Ricaricare l'ultimo file salvato ed aggiungere /tmp/track11.mp3come seconda traccia. Salvare con il menu File -> Dual audio avi.

    Ripetere per ogni spezzone.
