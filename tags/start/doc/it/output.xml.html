Formati di uscita

1.Formati di file

    Formato Restrizioni
    Avi Fino a 2 tracce audio
    Mpeg1/2 Flusso elementare grezzo
    Raw Audio o video stripped. Per esempio flussi mpeg4 grezzi.

2.Codec video

    Nome Commento Dipendenze
    Divx Opzioni non numerose, lento Divx
    Xvid Buono, accurato, veloce, ricco di opzioni Xvid
    Lavcodec Buono, accurato, veloce, ricco di opzioni Lavcodec (compreso)
    Mjpeg Mjpegtools
    H263 Configurazione lavcodec Compreso
    Huffyuv YV12 Huffyuv, non compatibile con Windows Compreso
    VCD/DVD/SVCD Buona qualità, lento mjpegtools
    XVCD Veloce, scarso, configurazione lavcodec Compreso
    Configurazione lavcodec significa che è necessario selezionare lavcodec (ffmpeg) come codec di uscita e configurarlo da lì.

3.Codec audio

    Nome Commento Dipendenze
    Uncompressed Voluminoso
    MP3 Buona qualità, solo CBR liblame
    MP2 Solo CBR, compatibile? Compreso (lavcodec)
    AC3 Solo CBR only, compatibile? Compreso (lavcodec)

4.Formati specifici

Avi 

    E' il formato predefinito di salvataggio. E' possibile aggiungere una seconda traccia audio CBR con misc->2nd audio track. Può essere indifferentemente MP3 CBR o AC3.

File Mpeg 

    Verrè scritto solamente un flusso elementare grezzo, .m1v o .m2v.

    Per ottenere un file audio .mp2 selezionare l'encoder audio mp2, configurare l'audio come process e selezionare File->save audio.

    Infine utilizzare mplex o tcmplex per effettuare il multiplex dei due file ed ottenere un .mpg valido.

    L'esportazione XVCD utilizza l'encoder e la configurazione lavcodec. per cvonfigurarlo selezionare il codec d'uscita FFmpeg ed utilizzare il menu configure video codec. Altamente sperimentale.

    I frame video Mpeg sono visti internamente come enormi frames YV12 non compressi. Perciò assicurarsi sempre di codificarli prima di salvarli o di utilizzare il salvataggio in forma grezza; NON salvare avi in modalità copia.

Raw (Grezzo) 

    Disponibile in due forme: per l'audio scegliere il menu File->save audio. Salverà solamente l'audio (processato/codificato o meno).

    Per il video utilizzare File ->save raw. E' utile per tagliare file mpeg o salvare video stripped, per esempio h263 grezzi o flussi mp4.
