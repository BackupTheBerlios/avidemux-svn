GUI Tour

1.Lower Toolbar

Frame type 

    Type of current image (I/B/P)

Play 

    Press it one time to play, a second time to stop

Pause 

    Press it to stop playback

Previous frame 

    Press it to go to the previous frame. It can take several seconds as avidemux may have to go back to previous I-frame.

Next frame 

    Press it to go to the next frame.

Previous Intra-frame 

    Press it to go to the previous intra frame (aka keyframe).

Next Intra-frame 

    Press it to go to the next intra frame (aka keyframe).

Markers 

    You see where the A(begin) and B(end) markers are. Only the area [A,B[ will be saved/processed..

Set marker A 

    Press it to set the A marker(see Cutting).

Set marker B 

    Press it to set the B marker(see Cutting).

Go to the beginning 

    Press it to go to the first frame

Go to the end 

    Press it to go to the last frame

Go to next "black" frame 

    Press it to go to next black frame, black frame being a bit subjective :)

2.Left Toolbar

Video filters 

    Open the video filter dialog box. See more about filters here

Audio filters 

    Open the audio filter dialog box. See more about filters here

Video codec button  

    Press this button to open the video codec selection/configure dialog.

Audio codec button  

    Press this button to open the audio codec selection/configure dialog.

VProcess button  

    When pressed, the saved video will be the processed video (i.e. with filters and encoding applied). If not pressed, it will be the raw copy of source video without alteration.

Audio process button  

    When pressed, the saved audio will be the processed audio (i.e. with filters and encoding applied). If not pressed, it will be the raw copy of source audio without alteration.

Preview button  

    Press this button to open a new window that shows the ouput of filters. If you navigate in the main window, the preview window will reflect the changes.

    This preview mode is terribly slow.

Ouput button  

    When pressed, the main window will show you the ouput of filters when playing, and only when playing.

    It is a fast display method, and in most cases you will have real time effects.

Info

Save
