Dual audio track

1.Avertissement

    Avidemux ne peut traiter en interne qu'une seule piste audio.

    Cependant, il est possible d'ajouter une seconde piste audio, avec certaines restrictions et notamment qu'aucun traitement ne sera possible sur cette piste supplémentaire.

2.Ajouter une seconde piste audio

Restrictions 

    La seconde piste audio doit être CBR : MP3 CBR ou AC3. Point barre.

Menu 

    Pour ajouter une seconde piste audio, utilisez le menu Misc->Second Track->xxxx. Après l'avoir sélectionnée, choisissez File -> Dual audio avi au lieu de File -> save avi.

Découper un avi avec deux pistes audio 

    Comme Avidemux ne peut pas traiter plus d'un flux audio en interne, il va falloir ruser un peu.

    Ouvrez l'avi, puis sélectionnez la deuxième piste audio lorsque le programme vous le demande.

    Définissez l'emplacement des marqueurs A et B et choisissez File -> save audio avec /tmp/track11.mp3 comme nom de fichier par exemple.

    Réouvrez le même fichier, mais choisissez cette fois-ci la première piste audio.

    Découpez-le puis enregistrez-le sous /tmp/avi1.avi.

    Rechargez-le et ajoutez /tmp/track11.mp3 comme seconde piste sonore. Enregistrez le résultat avec File -> Dual audio avi.

    Répétez ces opérations pour les différentes parties.
