4.Spécificités des formats

Avi 

    C'est le format d'enregistrement par défaut. Il est possible d'ajouter une seconde piste audio CBR avec la commande misc->2nd audio track. Le fichier choisi pour cette seconde piste peut être soit un MP3 CBR soit un AC3.

Fichier Mpeg  

    Seul le flux vidéo élémentaire brut est enregistré, avec pour extension .m1v ou .m2v.

    Il vous faudra alors sélectionné un encodage audio en mp2, passer le mode audio sur process et choisir File->save audio pour obtenir un fichier .mp2.

    Utilisez ensuite mplex ou tcmplex pour multiplexer les deux fichiers et obtenir un .mpg correct.

    L'export XVCD utilise l'encodeur et les réglages de lavcodec. Pour le configurer, sélectionez FFmpeg comme codec de sortie puis utilisez le menu configure video codec. C'en est encore au stade expérimental.

    D'un point de vue interne, les images d'une vidéo Mpeg apparaissent comme d'énormes images YV12 non compressées. Assurez-vous donc de les encoder avant d'enregistrer votre vidéo, ou utilisez les options d'enregistrement brut, mais N'ENREGISTREZ PAS un avi en mode copy.

Raw 

    Il existe sous deux formes : pour l'audio, il faut utiliser la commande File->save audio. Cela permet de n'enregistrer que l'audio (traité/encodé ou pas).

    Pour la vidéo, il faut utiliser File ->save raw. Cela peut être utile pour découper des fichiers mpeg ou pour enregistrer la vidéo séparément, comme un flux vidéo brut h263 ou mp4.
