Видео с двумя аудио дорожками

1.Внимание!

    Avidemux может управлять только одной аудио дорожкой внутри.

    Однако существует способ добавить вторую аудио дорожку, правда с некоторыми ограничениями .

2.Добавление второй аудио дорожки

Ограничения 

    Звук для второй дорожки должен быть записан как CBR, т.е. MP3 CBR или AC3.

Меню 

    Для добавления второй аудио дорожки используйте меню Misc->Second Track->xxxx . Затем при сохранении используйте File -> Dual audio avi вместо File -> save avi.

Вырезание в avi с двумя audio 

    Так как Avidemux не может управлять более чем одним звуковым потоком одновременно, в этом случае придется немного повозиться.

    Откройте avi и, когда будет предложено, выберите вторую звуковую дорожку.

    Установите маркеры и используйте File -> save audio как /tmp/track11.mp3 например.

    Откройте тот же avi, но в этот раз выберите первую дорожку.

    Вырежите необходимое и сохраните как /tmp/avi1.avi.

    Теперь откройте /tmp/avi1.avi и добавьте /tmp/track11.mp3 как вторую дорожку. Используйте диалог File -> Dual audio avi .

    Повторить для каждой части.
